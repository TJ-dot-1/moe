# Moenviron | Redefining Textile Waste

Moenviron is a modern, responsive web application that showcases scalable solutions to transform post-consumer textiles into reusable, recyclable, and traceable resources. This project serves as a dynamic corporate landing page bridging global supply chains with Africa’s growing circular economy.

## 🌟 Features

- **Futuristic & Clean UI:** A modern aesthetics featuring glassmorphism, dynamic glowing orbs, smooth transitions, and a curated forest green palette.
- **Fully Responsive:** Custom layout that adapts seamlessly to all devices (Mobile, Tablet, Desktop) using CSS Grid and Flexbox.
- **Interactive Animations:** Features a complex CSS-driven spinning dial in the hero section and scroll-triggered fade-up animations.
- **Semantic Structure:** Modular HTML layout categorized into distinct sections (Problem, Solution, How It Works, Africa's Role, Global Alignment, CTA).

## 🛠️ Technology Stack

- **HTML5:** Semantic styling and structured layout
- **CSS3:** Custom utility classes, native CSS variables, flex/grid layouts, and advanced keyframe animations.
- **JavaScript (Vanilla):** Mobile menu toggling, smooth scrolling, and scroll-based intersection observer logic.

## 🚀 Getting Started

Since this is a vanilla HTML/CSS/JS project, no build tools or package managers are required.

1. Clone the repository:
   ```bash
   git clone https://github.com/TJ-dot-1/moe.git
   ```

2. Navigate into the project folder:
   ```bash
   cd moe
   ```

3. Open the project in your browser:
   - Simply double-click `index.html` to open it locally.
   - Or, for a better development experience, use a local server like VS Code's **Live Server** extension.
